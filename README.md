Stock Broker CLient Web Dashboard
A simple Stock Broker Client Web Dashboard built using  JavaScript.
This project simulates live stock price updates and allows each user to subscribe to different stocks.
No backend or server is required â€” everything runs inside the browser.

Features

. Login with Email

. Subscribe / Unsubscribe to Stocks

. Live Stock Price Updates Every Second

. Real-time Sync Between Multiple Tabs (using BroadcastChannel API)

. Supports Multiple Users

. Subscriptions Saved per User (localStorage)

Supported Stocks
. GOOG  
. TSLA  
. AMZN  
. META  
. NVDA
